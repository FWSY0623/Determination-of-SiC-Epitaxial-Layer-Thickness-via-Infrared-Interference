# 碳化硅外延层厚度红外干涉测量

&emsp;&emsp;本项目基于2025年高教社杯全国大学生数学建模竞赛题目B，利用红外干涉光谱数据，建立双光束干涉与多光束干涉模型，反演碳化硅（SiC）及硅（Si）外延层厚度。针对折射率随波长变化的难题，通过柯西色散公式、非线性优化、快速傅里叶变换（FFT）和遗传算法（GA）实现高精度厚度估计。

## 一、问题背景

&emsp;&emsp;碳化硅是第三代半导体材料，外延层厚度很关键。红外干涉法通过看反射光谱的干涉条纹来测厚度，但有两个麻烦：一是折射率随波长变化，不能当成常数；二是当外延层和衬底的折射率差别不大时，多次反射造成的多光束干涉会影响结果。我们分三部分来处理：前两个问题只用双光束干涉，第三个问题考虑多光束干涉。

## 二、模型和方法

**1.双光束干涉（问题1和2）**  
&emsp;&emsp;只考虑上表面和界面的两次反射。利用干涉条件得到相邻极值点的波数差与厚度的关系。折射率随波数变化用三项柯西公式 $n_1(\nu) = A + B\nu^2 + C\nu^4$。调整 $A,B,C$ 让各极值点算出的厚度方差最小，用 MATLAB 的 `fmincon` 做非线性约束优化。

### 2. 多光束干涉（问题3）
考虑多次反射。把光分成 \( s \) 偏振和 \( p \) 偏振，用菲涅尔公式和斯托克斯关系推出非偏振光的总反射率 \( R \) 与波数 \( \nu \) 的关系：

$$
R = \frac{1}{2} \left( 
\left| \frac{r_{01,s} + r_{12,s} e^{i\delta}}{1 + r_{01,s} r_{12,s} e^{i\delta}} \right|^2 + 
\left| \frac{r_{01,p} + r_{12,p} e^{i\delta}}{1 + r_{01,p} r_{12,p} e^{i\delta}} \right|^2 
\right),
$$

其中 $\delta = 4\pi d n_1 \nu \cos\theta_1$.

&emsp;&emsp;我们用了四步反演：先用FFT粗估厚度，再假设 $n_1 \approx 1.1n_2$ 逐点反演折射率，然后用柯西公式拟合 $n_1(\nu),n_2(\nu)$，最后用遗传算法（GA）全局优化厚度 $d$。判断多光束干涉的依据是光强比 $\eta$ 和条纹形态（FFT频谱是否尖锐）。

## 三、主要结果

- 双光束模型（SiC 外延层）：厚度 $7.4465\times10^{-4}$ cm，变异系数6.83%，参数扰动不敏感。
- 多光束模型（Si 外延层）：厚度 $3.2723\times10^{-4}$ cm，理论反射率和实验数据吻合不错。

&emsp;&emsp;附件1和2（SiC）的多光束干涉可以忽略，双光束模型就够了；附件3和4（Si）必须用多光束模型，遗传算法避免了局部最优。

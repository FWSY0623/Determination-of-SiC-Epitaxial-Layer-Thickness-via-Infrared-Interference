# attachment文件夹说明

&emsp;&emsp;本文件夹存放了题目提供的原始光谱数据。

## 文件列表

- `附件1.xlsx`：碳化硅晶圆片，入射角10°  
- `附件2.xlsx`：碳化硅晶圆片，入射角15°  
- `附件3.xlsx`：硅晶圆片，入射角10°  
- `附件4.xlsx`：硅晶圆片，入射角15°  

## 数据格式

&emsp;&emsp;每个Excel文件有两列。第1列为**波数（单位：cm⁻¹）** ，第2列为**反射率（单位：%）**。

## 注意事项

- 附件1和2用于双光束干涉模型（碳化硅外延层厚度计算）。  
- 附件3和4用于多光束干涉模型（硅外延层厚度计算）。  
- 读取数据时建议只使用波数> 2000 cm⁻¹的部分（高波数区域干涉条纹稳定，符合模型假设）。  

&emsp;&emsp;详细处理方法参见根目录的 `src/` 中的MATLAB代码。
